# Changesets

Este diretório é usado pelo [Changesets](https://github.com/changesets/changesets)
para versionar os pacotes públicos (`@botozap/sdk`, `@botozap/cli`, `@botozap/mcp`).

- Nova mudança que afeta um pacote publicável: `pnpm changeset` e descreva.
- Versões seguem `0.x` até decisão explícita de estabilidade (ver roadmap E3).
- A publicação no npm NÃO é automatizada nesta fase (E3.2); acontece na E3.3,
  manualmente, após os gates de qualidade.
